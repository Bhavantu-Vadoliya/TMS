# TMS
# Task Management System

## Description
This project is an academic project designed to manage tasks and leave applications efficiently. It provides a user-friendly interface for employees and managers to handle their tasks and leave requests.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Bhavantu-Vadoliya/TMS.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TMS
   ```
3. Install any required dependencies (if applicable).

## Database Setup
1. Import the `tms.sql` file located in the `Database` directory to your MySQL database.
2. Update the database connection settings in `Source code/TMS/includes/config.php` if necessary.

## Usage
To start the application, run the following command:
```bash
php -S localhost:8000 -t Source code\ TMS
```
Then, open your browser and navigate to `http://localhost:8000`.
